## Описание проекта

**Домовой** - это современный веб-сайт для риэлторского агентства, созданный для упрощения поиска недвижимости и взаимодействия клиентов с профессиональными агентами.

### Проблема, которую решает проект:
- Сложность поиска подходящей недвижимости вручную

- Отсутствие удобных инструментов фильтрации объектов

### Основные цели:
- [x] Создать интуитивно понятный интерфейс поиска недвижимости
- [x] Предоставить подробную информацию об объектах и агентах
- [x] Обеспечить адаптивность для всех устройств
- [x] Реализовать форму обратной связи для клиентов

Описание проекта

Краткое описание:
"Домовой" — современный веб-сайт для риэлторского агентства, созданный для упрощения поиска недвижимости и взаимодействия клиентов с профессиональными агентами. Это одностраничное приложение предоставляет удобный интерфейс для поиска объектов недвижимости с фильтрацией, просмотра информации об агентах и отправки заявок на консультацию. Сайт полностью адаптивен и работает на всех устройствах от смартфонов до десктопов.

Проект реализован в рамках практической работы по верстке сайта по предметной области "Риэлторское агентство". Он использует чистые технологии HTML5, CSS3 и JavaScript без сторонних фреймворков, что обеспечивает быструю загрузку и высокую производительность.

Проблема, которую решает проект:
  Сложность поиска недвижимости — многие платформы имеют неудобные интерфейсы поиска, что затрудняет клиентам нахождение подходящих объектов

  Отсутствие доверия к агентам — пользователям сложно оценить профессионализм и надежность агентов без подробной информации и контактов

  Неадаптивность существующих решений — многие сайты недвижимости плохо отображаются на мобильных устройствах

Основные цели:
  Создать интуитивно понятный интерфейс поиска недвижимости с расширенными фильтрами
  
  Реализовать адаптивный дизайн, корректно отображающийся на всех типах устройств

  Предоставить полную информацию об агентах с возможностью прямого контакта

  Добавить систему личных кабинетов для сохранения избранных объектов

  Интегрировать карты для визуализации расположения объектов

  Реализовать систему онлайн-записи на просмотр недвижимости
