Часто задаваемые вопросы
Общие вопросы
❓ Что это за проект?

Это "Домовой" - веб-сайт риэлторского агентства, предназначенный для поиска недвижимости, получения информации об объектах и связи с профессиональными агентами.
❓ Кто может использовать этот проект?

Проект предназначен для:

  Покупателей недвижимости - для поиска жилья по заданным параметрам

  Продавцов недвижимости - для размещения информации о своих объектах

  Арендаторов - для поиска жилья в аренду

  Инвесторов - для анализа рынка недвижимости

  Риэлторов - в качестве шаблона для собственного сайта

❓ Нужно ли платить за использование?

Нет, проект полностью бесплатный и имеет открытый исходный код под лицензией MIT. Вы можете использовать, модифицировать и распространять его бесплатно.
Технические вопросы
⚙️ Какие системные требования?

  Браузеры: Chrome 80+, Firefox 75+, Safari 13+, Edge 80+

  Интернет-соединение: требуется для загрузки шрифтов и иконок

  Разрешение экрана: оптимально 1280x720 и выше, но адаптировано и для меньших разрешений

  JavaScript: должен быть включен в браузере

  Память: минимум 512MB ОЗУ

🔧 Как установить проект?

Следуйте инструкциям в разделе Установка и запуск документации. Основные шаги:

  Клонировать репозиторий: git clone https://github.com/ваш-username/домовой-риэлторство.git

  Открыть файл index.html в браузере

🐛 Как сообщить об ошибке?

Создайте Issue на GitHub с подробным описанием проблемы:

  Перейдите в раздел Issues репозитория

  Нажмите "New Issue"

  Заполните шаблон:

  Описание проблемы: что происходит?

  Шаги воспроизведения: как вызвать ошибку?

  Ожидаемое поведение: что должно происходить?

  Фактическое поведение: что происходит на самом деле?

  Скриншоты: если возможно, приложите скриншоты

  Окружение: ОС, браузер, версия

Вопросы по разработке
💻 Как внести вклад в проект?

Прочитайте CONTRIBUTING.md для получения инструкций. Основные этапы:

  Форкнуть репозиторий

  Создать ветку для новой функциональности

  Внести изменения и протестировать их

  Создать Pull Request с описанием изменений

📄 Где найти документацию API?

См. раздел API документация. В текущей версии проект не использует внешние API, но в планах:

  Интеграция с картографическими сервисами

  Подключение к базам данных недвижимости

  REST API для управления объектами

🔐 Как работает авторизация?

В текущей версии авторизация не реализована, так как сайт является статическим. В планах развития:

  JWT-токены для аутентификации

  OAuth 2.0 для входа через социальные сети

  Ролевая модель (администратор, агент, клиент)

Вопросы по функциональности
🏠 Как добавить новые объекты недвижимости?

В текущей версии объекты добавляются путем редактирования HTML-кода в файле index.html. Найдите раздел с классом offers__grid и добавьте новую карточку по аналогии с существующими.
📱 Поддерживаются ли мобильные устройства?

Да, сайт полностью адаптивен и оптимизирован для:

  Смартфонов (iOS и Android)

  Планшетов (iPad, Android-планшеты)

  Десктопных компьютеров

  Ноутбуков с разными разрешениями экрана

🔍 Можно ли изменить цветовую схему?

Да, цветовая схема задана через CSS-переменные в начале файла index.html (внутри тега <style>). Измените значения переменных:
css

:root {
    --color-primary: #1E3A8A; /* Основной синий */
    --color-accent: #D4A017;  /* Акцентный золотой */
    /* ... остальные цвета */
}

📊 Как добавить новых агентов?

Для добавления новых агентов:

  Найдите раздел с классом agents__grid в index.html

  Добавьте новую карточку агента по аналогии с существующими

  Обновите информацию (имя, фото, контакты, описание)
